# Mohamed Khaled — Portfolio Website

A responsive one-page portfolio website built with plain HTML, CSS, and JavaScript.

## Files
- `index.html` — the full portfolio website.
- `assets/Mohamed_Khaled_Flutter_CV.pdf` — downloadable CV linked from the portfolio.

## How to preview it
Open `index.html` directly in a browser.

## How to publish it for free with GitHub Pages
1. Create a new GitHub repository, for example `mohamed-khaled-portfolio`.
2. Upload all files in this folder, preserving the `assets` folder.
3. In the GitHub repository, open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/ (root)`, then save.
6. GitHub will provide a public portfolio URL within a few minutes.

## Before publishing
- Verify the LinkedIn URL and phone number in `index.html`.
- Add real screenshots of the projects later if you have permission to share them. The current project visuals are custom abstract UI illustrations and do not expose confidential client data.
